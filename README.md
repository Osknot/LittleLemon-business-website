# Little Lemon Business Website

### Meta Back-End Developer Professional Certificate - Capstone Project

---

## PROJECT OVERVIEW

This repository contains the final project for the **Coursera Django Web Framework** course. The application is a full-stack web solution for **Little Lemon**, a Mediterranean restaurant, allowing users to view the menu, learn about the establishment, and book tables online.

The project focuses on implementing the **MVT (Model-View-Template)** architecture, handling database relations, and securing the application with Django's authentication system.

---

## KEY FEATURES

- **DYNAMIC MENU**
  Menu items are stored in a SQLite database and rendered dynamically using Django Template Language (DTL).

- **BOOKING SYSTEM**
  A functional reservation form that validates user input and saves booking details to the backend.

- **CUSTOM ADMIN INTERFACE**
  The Django Admin panel is configured to allow restaurant staff to manage menu content and reservations easily.

---

## TECH STACK

- **Language:** Python 3.x
- **Framework:** Django 4.x
- **Database:** SQLite3
- **Frontend:** HTML5, CSS3 (including responsive design for mobile-first access)
- **Tools:** Pipenv/Virtualenv for dependency management
